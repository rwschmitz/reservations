# Reservations

## Description

This is a React Native application built to handle reservations.

Features included are:

- Allow users to create a reservation.
- Allow users to view existing reservations.

Code organization:

```bash
├── app
│   ├── components
│   │   ├── __tests__
│   ├── queries
│   ├── screens
│   └── styles
├── LICENSE
├── CHANGELOG
├── README.md
├── .gitignore
└── .markdownlint.json
```
